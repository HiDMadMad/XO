this is a simple XO(tic tac toe) game that you can run and enjoy instantly.

it runs smoothly inside VSCode and there is also an executable version if you just want to jump in and play.
but it does not have any special abilities or fancy features. simple and raw.
the AI is completely random and honestly a little bit dumb. it does not calculate, it does not plan ahead, it just picks a move and hopes for the best.

if you feel like making it better by adding features, improving the AI or just experimenting; ideas, issues, and pull requests are always welcome.
it would be cool to work on it together and improve it step by step.
